破坏球

重装

英雄简介

破坏球是一只名叫哈蒙德（Hammond）的仓鼠，驾驶着一台名为"破坏球"的自制战斗机甲。他是地平线月球基地的"实验体"，与温斯顿一样拥有被基因工程增强的智力。破坏球具有极高的机动性，可以变形成球形快速滚动，用抓钩在环境中摆荡，部署自适应护盾保护自己，并用四联机炮和地雷清扫场造成巨大伤害。他是游戏中最具机动性的坦克英雄，擅长扰乱敌方阵型和创造混乱。

技能介绍

01

四联机炮

破坏球的主要武器，四联装自动机炮，每发造成5点伤害，射速快但弹道扩散较大。适合中近距离作战，在近距离可以造成可观的伤害。弹药无限，但需要冷却（过热机制）。

02

球形模式

破坏球的核心技能，将机甲变形为球形，大幅提升移动速度。在球形模式下可以攀爬斜坡和墙壁，使用抓钩摆荡，并撞击敌人造成伤害和击退效果。

03

抓钩

破坏球发射抓钩附着在表面上，然后可以摆荡或将自己拉向目标。是破坏球实现高机动性的关键技能，可以用来快速移动、逃离危险或撞击敌人。没有冷却时间，但需要表面附着点。

04

地雷清扫场

破坏球的终极技能，在周围部署一圈感应地雷。地雷会在敌人接近时爆炸，造成130点伤害。地雷可以被摧毁，但数量众多，可以有效地控制区域或逼迫敌人走位。持续20秒。

被动技能

P1

自适应护盾

破坏球激活自适应护盾，根据附近敌人的数量获得临时护盾。每有一个敌人获得100点护盾，最多获得700点护盾。护盾持续7秒，是破坏球重要的生存技能。冷却时间15秒。

P2

动力坠击

破坏球从空中撞击地面，对范围内的敌人造成伤害并击飞。伤害取决于坠落高度，最高100点。可以配合抓钩从高处坠落，是重要的控制技能。

背景故事

哈蒙德是一只来自地平线月球基地的基因工程仓鼠，与温斯顿一样是"地平线月球殖民地"的实验体。当温斯顿启动逃生舱返回地球时，哈蒙德也偷偷建造了自己的逃生舱，并跟随温斯顿来到了地球。

与温斯顿不同，哈蒙德对人类世界没有崇高的理想。他降落在澳大利亚的渣客镇，这个充满混乱和暴力的废土城市。哈蒙德凭借自己的智慧和工程技能，从渣客镇的废料中建造了一台战斗机甲，并称之为"破坏球"。

哈蒙德很快在渣客镇的机甲大战中崭露头角。他的小型体型和快速移动的球形机甲让其他参赛者难以应对。他赢得了多场机甲大战，积累了财富和名声。

尽管哈蒙德是一只仓鼠，但他的智力远超普通人类。他不仅精通机械工程，还懂得战术和策略。他将自己的机甲设计得既能快速移动，又能造成巨大破坏，完美结合了机动性和火力。

哈蒙德最初参加机甲大战只是为了生存和财富，但随着时间的推移，他开始享受这种战斗带来的刺激感。他将每一场战斗都视为一场游戏，总是带着轻松愉快的心态参与。

现在，哈蒙德驾驶着他的破坏球机甲在世界各地参加机甲比赛和战斗。他没有像温斯顿那样加入守望先锋，而是选择了一种更自由、更刺激的生活方式。他相信，生活就应该充满乐趣和冒险。

尽管外表看起来只是一只可爱的仓鼠，但哈蒙德是一名出色的战士和工程师。他用行动证明，即使是小型生物也能成为强大的力量。

"游戏开始！"——哈蒙德

使用技巧

1. 机动性最大化：破坏球的优势在于机动性。不断使用球形模式和抓钩移动，让敌人难以瞄准。记住，抓钩没有冷却时间，只要找到附着点就可以使用。

2. 撞击连招：破坏球的经典连招是：使用抓钩摆荡获得速度 → 撞击敌人造成伤害和击退 → 使用动力坠击再次造成伤害和击飞 → 变形为站立模式用机炮攻击。

3. 自适应护盾时机：自适应护盾根据附近敌人的数量提供护盾。在冲入敌阵时使用，可以获得最大护盾值。护盾持续7秒，足够时间造成混乱后撤离。

4. 地雷清扫场使用：地雷清扫场是强大的区域控制技能。最佳使用时机包括：控制目标点、封堵敌人撤退路线、配合队友控制技能使用。注意地雷可以被摧毁，尽量放在敌人难以瞄准的位置。

5. 目标优先级：优先攻击支援英雄和脆皮输出英雄。破坏球不适合正面作战，更适合骚扰后排、打断敌方支援和治疗。

6. 生存策略：破坏球虽然有高机动性，但生命值不高。学会"打带跑"战术：冲入敌阵造成混乱 → 造成伤害 → 使用自适应护盾 → 快速撤离 → 等待技能冷却 → 再次攻击。

战斗定位

骚扰型坦克：破坏球是典型的骚扰型坦克，不适合站在前排吸收伤害，而是通过高机动性扰乱敌方阵型。

后排杀手：凭借高机动性和控制能力，破坏球可以快速突入敌方后排，击杀或驱赶支援和输出英雄。

控制专家：撞击和动力坠击是强大的控制技能，可以打断敌方技能、击飞敌人、创造击杀机会。

区域控制者：地雷清扫场可以有效地控制大片区域，逼迫敌人改变位置或清理地雷。

信息收集者：破坏球的高机动性让他可以快速侦察敌方位置和阵型，为团队提供宝贵信息。

电竞技巧

1. 抓钩高级技巧：顶级玩家会使用抓钩实现复杂移动，如墙上奔跑、快速转向、空中变向等。掌握这些技巧可以大幅提升破坏球的机动性。

2. 能量管理：破坏球的终极技能充能很快。在高端局中，破坏球玩家应该积极造成伤害和控制，快速充能地雷清扫场。

3. 撞击时机：撞击不仅可以造成伤害，还可以打断关键技能。学会预判敌方英雄的技能释放时机，用撞击打断它们。

4. 地雷布置艺术：地雷的布置位置很重要。最佳位置包括：狭窄通道、目标点中心、敌人必经之路。也可以将地雷布置在空中，让敌人难以摧毁。

5. 团队配合：破坏球需要与团队配合。告诉队友你的攻击目标，配合队友的控制技能使用地雷清扫场，与另一个骚扰英雄协同作战。

6. 地图掌握：破坏球可以到达许多其他英雄无法到达的位置。熟悉地图的每一个角落、每一个斜坡和墙壁，最大化利用机动性优势。
